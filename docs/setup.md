# Editing Markdown Files / Édition des fichiers Markdown

## 🇫🇷 Installer MarkText (pour éditer les fichiers facilement)

MarkText est un éditeur simple et gratuit pour écrire en français sans vous soucier de la syntaxe markdown.

**Étape 1 — Télécharger MarkText**
1. Allez sur **[marktext.app](https://marktext.app)** (site officiel)
2. Cherchez le bouton **"Download"** (télécharger)
3. Choisissez votre système d'exploitation :
   - **Mac** → cliquez sur "macOS"
   - **Windows** → cliquez sur "Windows"
4. Ouvrez le fichier téléchargé et suivez l'installateur

**Étape 2 — Utiliser MarkText pour éditer**
1. Ouvrez MarkText
2. Allez dans **File** (Fichier) → **Open** (Ouvrir)
3. Naviguez jusqu'au dossier `copywriting` et sélectionnez le fichier que vous voulez éditer (par exemple, `setup.md`)
4. Tapez comme vous le feriez dans Word — MarkText s'occupe du formatage

**Astuce :** Vous pouvez laisser MarkText ouvert sur le côté de votre écran et Claude Code de l'autre, pour pouvoir basculer rapidement entre l'édition et le chat.

## 🇬🇧 Installing MarkText (to edit files easily)

MarkText is a simple, free editor for writing without worrying about markdown syntax.

**Step 1 — Download MarkText**
1. Go to **[marktext.app](https://marktext.app)** (official website)
2. Look for the **"Download"** button
3. Choose your operating system:
   - **Mac** → click "macOS"
   - **Windows** → click "Windows"
4. Open the downloaded file and follow the installer

**Step 2 — Use MarkText to edit**
1. Open MarkText
2. Go to **File** → **Open**
3. Navigate to your `copywriting` folder and select the file you want to edit (e.g., `setup.md`)
4. Type as you would in Word — MarkText handles the formatting

**Tip:** You can keep MarkText open on one side of your screen and Claude Code on the other, so you can flip between editing and chatting easily.
