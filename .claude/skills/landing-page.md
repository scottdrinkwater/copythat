# Skill: Landing Page Copy Writer

Use this skill when the user needs to write or structure a landing page, sales page, or homepage.

## Page structure (proven sequence)

Use this section order as the default. Adjust based on page type and user input.

```
1. HERO
   - Headline: outcome-led, specific, instantly clear what it's for
   - Subheadline: adds context or addresses the sceptic
   - CTA button: action verb + outcome (not just "Submit")
   - Optional: trust signal (e.g. "Join 4,000 marketers" or a logo bar)

2. PROBLEM / EMPATHY
   - Name the frustration the reader is feeling right now
   - Make them feel seen before you pitch
   - 2–4 short paragraphs or bullet points

3. SOLUTION / PRODUCT INTRO
   - Introduce what you offer as the answer
   - Lead with transformation, not features
   - One clear sentence: "X is [what it is] that helps [who] [achieve what]"

4. FEATURES → BENEFITS
   - For each feature: explain what it means for the user
   - Format: "[Feature name] — [What it does for you]"
   - Aim for 3–6 items

5. SOCIAL PROOF
   - Testimonials: pull a specific result, not a vague compliment
   - Case study snippet: before/after, with a number if possible
   - Logos or stats if available

6. OBJECTION HANDLING / FAQ
   - Address the 3–5 most common reasons people don't buy
   - Keep answers short and honest
   - A good FAQ doubles as a persuasion layer

7. PRICING / OFFER (if applicable)
   - Make the value obvious before the price
   - Frame with what they're getting, not what they're paying

8. FINAL CTA
   - Restate the core benefit
   - One clear action
   - Urgency or guarantee if genuine
```

## Writing principles for landing pages
- Write to one person, not a crowd — use "you" throughout
- Every section should earn the scroll — if it doesn't convert, cut it
- The best copy sounds like a conversation, not a brochure
- Avoid vague words: "powerful", "seamless", "innovative" — use specifics
- Subheadings should work as a standalone summary of the page argument

## Output format
Write each section with a clear heading label, e.g.:

```
[HERO]
Headline: ...
Subheadline: ...
CTA: ...

[PROBLEM]
...
```

After writing, offer: "Want me to tighten any section, adjust the tone, or add more social proof language?"

## Inputs to ask for (if not provided)
- What is the product or service?
- Who is the audience? (the more specific the better)
- What is the main CTA / conversion goal?
- Is there an existing brand voice guide to follow?
- Any specific objections or doubts the audience tends to have?
- Are there testimonials, case studies, or stats to include?
