# Skill: Ad Copy Writer

Use this skill when the user needs copy for paid ads — Google, Meta (Facebook/Instagram), LinkedIn, or display.

## Trigger phrases
- "ad copy", "write an ad", "ads", "paid copy"
- "Google ad", "Facebook ad", "Instagram ad", "LinkedIn ad"
- "display ad", "banner copy"

## Platform formats and constraints

### Google Search Ads
```
Headlines: up to 15 × max 30 characters each (at least 3 required)
Descriptions: up to 4 × max 90 characters each (at least 2 required)

Best practice:
- Include the keyword naturally in at least one headline
- Lead with the benefit, not the brand
- Use sentence case, not ALL CAPS
- Include a clear CTA in descriptions ("Get a free quote", "Start free today")
- Pin Headline 1 = main benefit, Headline 2 = differentiator, Headline 3 = CTA or offer
```

### Meta (Facebook & Instagram) Ads
```
Primary text: 125 characters before "See more" (aim to land the hook here)
Headline: max 40 characters (appears below image/video)
Description: max 30 characters (optional, appears below headline)

Formats to write for:
- Feed ad (primary text + headline + CTA)
- Story / Reel ad (punchy hook — first 3 seconds must stop the scroll)
- Carousel (short headline per card + single benefit each)

Best practice:
- Open with a hook: question, bold claim, relatable pain, or pattern interrupt
- Speak to one specific person, not a broad audience
- Social proof in primary text converts well ("Over 10,000 businesses use...")
- CTA should match the funnel stage (cold = "Learn more", warm = "Get started", hot = "Buy now")
```

### LinkedIn Ads
```
Sponsored Content:
- Introductory text: 150 chars before truncation (600 max)
- Headline: max 70 characters
- Description: max 70 characters

Best practice:
- Professional but not stiff — write like a smart colleague, not a press release
- Lead with a business outcome or data point
- Audience is problem-aware: skip the 101, go straight to the insight
- Works well for: lead gen, whitepaper downloads, webinar sign-ups
```

## How to execute

1. Confirm which platform(s) and ad format
2. Write multiple variations — at least 3 headline options, 2 description or body options
3. Flag character counts if tight
4. Label clearly: `[HEADLINE 1]`, `[PRIMARY TEXT]`, etc.

## Output format

```
PLATFORM: [Google / Meta / LinkedIn]
FORMAT: [Search / Feed / Story / Carousel / Sponsored Content]

[HEADLINE OPTIONS]
1. ... (XX chars)
2. ... (XX chars)
3. ... (XX chars)

[DESCRIPTION / PRIMARY TEXT OPTIONS]
A. ...
B. ...

[CTA]
Suggested: [button text]
```

## Inputs to ask for (if not provided)
- Platform and ad format
- Product or offer being advertised
- Target audience — who sees this? What do they want?
- Funnel stage — cold (awareness), warm (consideration), or hot (conversion)?
- Any offer, discount, or urgency to include?
- Landing page destination (helps match the message)
