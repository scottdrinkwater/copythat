# Skill: SEO Web Copy Writer

Use this skill when writing or editing website copy that needs to rank in search engines — service pages, location pages, homepages, about pages, and pillar content.

## SEO copy principles

### Keyword placement hierarchy
```
Title tag:      Primary keyword near the start (max 60 chars)
H1:             Primary keyword — can be reworded, not a copy-paste of title tag
First 100 words: Natural mention of primary keyword
H2s/H3s:       Secondary keywords, related terms, question-based phrases
Body:           Keyword variations, synonyms, semantic terms (LSI)
Meta description: Primary keyword + compelling reason to click (max 155 chars)
```

### Page structure for SEO
```
1. Title tag + H1 (primary keyword, clear topic signal)
2. Opening paragraph — state what the page is about, for whom, and why it matters
3. H2 sections — each targets a secondary keyword or related question
4. Proof / trust signals — testimonials, stats, accreditations (E-E-A-T signals)
5. FAQs — target "People Also Ask" queries; structured for featured snippet capture
6. CTA section — clear next step
7. Internal linking — suggest 2–3 pages to link to/from
```

### E-E-A-T signals to weave in
Google's quality framework: Experience, Expertise, Authoritativeness, Trustworthiness.
- Mention specific results, years of experience, qualifications, or methods
- Use first-person where appropriate ("We've worked with...")
- Cite real data points where possible
- Include a clear author or business entity

### Writing for scannability (also helps SEO)
- Short paragraphs (2–3 lines max)
- Descriptive subheadings that work without the surrounding text
- Bullet lists for features, benefits, or steps
- Bold key phrases (not entire sentences)

For each page, produce:

```
TITLE TAG: ... (XX/60 chars)
META DESCRIPTION: ... (XX/155 chars)

H1: ...

[OPENING PARAGRAPH]
...

[H2: ...]
...

[H2: ...]
...

[FAQ SECTION]
Q: ...
A: ...

Q: ...
A: ...

[CTA]
...

INTERNAL LINK SUGGESTIONS:
- Link from this page to: [page name / topic]
- Link to this page from: [page name / topic]
```

## Inputs to ask for (if not provided)
- Primary keyword to target
- Secondary keywords or related terms (if known)
- Page type (service, location, homepage, about, pillar)
- Target audience — who is searching for this?
- Business name, location (if local SEO)
- Competitors or pages to outperform (optional)
- Existing content to optimise (paste it in)
